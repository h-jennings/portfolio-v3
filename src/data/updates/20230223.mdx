Added syntax highlighting for code blocks for mdx files using
[rehype-pretty-code](https://rehype-pretty-code.netlify.app/)

It supports inline code `const test = () => console.log('hello world');{:ts}`

And code blocks:

```tsx
const test = () => {
  console.log('hello world');
};
```
