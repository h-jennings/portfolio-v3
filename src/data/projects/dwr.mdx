The F-35 Joint Strike Fighter Program spans decades, with data scattered across systems, teams, and timelines. Stakeholders needed a way to actually make sense of it.

The application uses interactive visualizations to surface patterns and give decision-makers a single source of truth. The design had to be as clear as the data was complex.
