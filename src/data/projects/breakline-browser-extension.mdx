BreakLine's sourcing team needed participant data accessible while browsing
LinkedIn. The extension runs as a Chrome sidepanel, detecting profiles
automatically and surfacing matches in real time.

Matching works in two passes: first by LinkedIn URL, then by name and metadata,
with keyboard-navigable search as a fallback. Built with WXT, React, and
TanStack Query on a custom AWS Amplify/Cognito auth layer. I owned it end-to-end
— architecture, CI/CD, and automated Chrome Web Store publishing.

Credit to [Kyle Kochanek](https://www.kocheck.dev/) for the product design.
