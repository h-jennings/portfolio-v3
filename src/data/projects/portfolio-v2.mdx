Every time I start a new personal project, I look for an excuse to learn
something I haven't done before. For this version of my portfolio, that was
motion design.

Using framer-motion, I built a suite of declarative, reusable animations in
React — page transitions, scroll-reveal effects, orchestrated sequences. The
motion became part of the design language of the site.
